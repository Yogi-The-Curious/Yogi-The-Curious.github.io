Abstract¶

The heat generation of a lithium-ion battery during charge/discharge can be iteratively modeled to gain insights into battery longevity, safety, and usage. This project explores the process of modeling battery temperature as a product of internal resistance, and convective cooling. An 18650 Lithium-Ion battery is used for modeling. The step by step modeling process will provide an accessible path to begin modeling lithium battery temperatures.

Intro¶

This project was of interest for its value in understanding the relationship between current,power sources, and thermal dynamics. This project will help myself and the reader to gain a basic understanding of the thermal factors at play in battery heating. Additionally the iterative modeling process detailed in this project provides a framework for future modeling and research endeavors.

First we ignored for the moment any cooling done. We used Joules equation to describe the power created from current and battery internal resistance. This applies to both charging and discharging currents.

 

$$ P = I^{2}R $$

 

By recalling that energy is power over time we rewrote Joules equation as:

 

$$ H = I^{2}Rt $$

 

By combining this increase in energy over time with a heat transfer equation we found a general formula to describe the change in battery temperature over time. This equation assumes a constant mass, current, and resistance.

 

$$I^{2}Rt = mc(T_{f}-T_{i})$$ $$I^{2}Rt = mcT_{f} - mcT_{i}$$ $$I^{2}R = \frac {mcT_{f} - mcT_{i}}{t}$$

 

To iteratively model this we needed to separate final battery temperature:

 

$$T_{f} = T_{i} + \frac{I^{2}Rt}{mc}$$

 

Our next step was to incorporate the effects of convective cooling into our equation. We began by looking at Newton's cooling equation.

 

$$ \frac{dT}{dt} = -k(T_{o}-T_{a})$$

 

Adding this to our simplest model:

 

$$T_{f} = (T_{i} + \frac{I^{2}Rt}{mc}) - (k(T_{i} - T_{a}))$$

 

This new equation added some constants that we did not have any data for. We were able to extract and estimate these from similar studies. (add links to those references later)

 

At this point it became clear from the data of similar studies (reference a few if I can) a charging current has nearly an order of magnitude lesser effect on temperature compared to typical discharge currents. From this conclusion we turned our attention to focus on modeling data from discharge currents. In our second iteration we modeled 1C and 5C currents.

In our next iterations we will focus on modeling the effects of : decreasing internal resistance, chemical enthalpy, and lumped capacitance model. We will also look to expand the output of our models by mapping the effects of varying initial conditions over time.

Attaining these additions would allow for more accurate predictive models covering a wider range of conditions. At such a time when a desired level of accuracy is obtained it will become useful to create an interactive model, providing a useful tool for electrical design.

Model One¶

To create our simplest model we wrote an iterative python program. We chose to iterate our calculations over one second intervals. We utilized equation (use labeled equation). # Purpose: To model Thermal Cooling and heating of a lithium ion battery during charging.
# written for a single sony 18650 cell

# --------------- Here we will list variables-----------------------
import sys
import numpy as np
import matplotlib.pyplot as plt
from decimal import *

getcontext().prec = 8

def solve_temp (current,time):
    ''' Inputs: current : decimal, time: interger 
        outputs:  returns numpy array of time and T_f '''

    # initializing variables
    r = Decimal(.013)  # Resistance in Ohms
    m = Decimal(.044)  # Mass of battery in kg
    t_i = Decimal(20)  # initial Temperature of Battery (room temp celsius)
    t_f = Decimal(0)  # Final temperature of Battery (will be solving for this )
    c = Decimal(902)  # Specific heat capacity in Joules per kg celsius
    time_array = np.array([])
    temp_array = np.array([])
    for t in range(1,time):
        step_1 = current ** 2
        step_2 = step_1 * r
        step_3 = step_2 / m
        step_4 = step_3 / c
        t_f = t_i + step_4
        t_i = t_f
        time_array = np.append(time_array, [t])
        temp_array = np.append(temp_array, [t_f])
        t+= 1

    return time_array, temp_array


time = 3600  # time (our iterable) in seconds
current =  [Decimal(1), Decimal(3.4)]  # Current in Amps recommend to be 1A charging and 3.4 Amps discharge
charging = solve_temp(current[0], time)
discharge = solve_temp(current[1],time)

plt.figure()

plt .subplot().set(xlabel = 'Time (s)', ylabel = 'T emp (C)', title = ' Temp vs Time (1 Hour)', )
plt.plot(charging[0], charging[1] , "g--", label = "charging")
#plt.subplot().set(xlabel = 'Time (s)', ylabel = 'Temperature (C)', title = ' Discharging',)
plt.plot(discharge[0],discharge[1],"r--",label = "discharging")
plt.legend()
plt.show()

Results & Discussion¶

Modeling recommended 3.4 A charge and 1 A discharge currents immediately shows the quadratic effect current has on battery temperature. This initial model provided baseline temperature changes for our following models. The linear behavior shown in the model would indicate the temperature constantly rising over time with no asymptotic behavior. This indicated the need for a cooling term to give a realistic behavior.

Model Two (Newtons Cooling)¶

The second model adds Newton's cooling equation to represent convective cooling. We assume a 25 Celsius ambient temperature and -10 C initial battery temperature, 1C and 5C discharge current. A quick note on battery discharge measurements: The C unit commonly used in battery studies is a measurment of how long it takes to discharge a battery in one hour. For the 18650 battery we used here 1C equates to a 2.5A current, as our battery is rated at 2500mAh.

These values were used to compare results of similar study "Internal Temperature Prediction of Lithium-ion Cell Using Differential Voltage Technique".

We were iterating over time, and so we shrank time by increasing our iterations. We therfore multiplied over the dt that is found in the orginial differential cooling equation. We can do this in our loop but not on a piece of paper because we are constantly updating the temperature of our battery with each increment of time. Something a human couldn't do. This is how we inserted the cooling term into our code. $$ T_{f} = T_{i} - k(T_{i} - T_{a})$$

# written for a single sony 18650 cell

# --------------- Here we will list variables-----------------------
import numpy as np
import matplotlib.pyplot as plt
from decimal import *

getcontext().prec = 8

def solve_temp (current,time):
    ''' Inputs: current : decimal, time: interger 
        outputs:  returns numpy array of time and T_f '''

    # initializing variables
    r = Decimal(.013)  # Resistance in Ohms
    m = Decimal(.044)  # Mass of battery in kg
    t_i = Decimal(-10) #initial Temperature of Battery
    t_f = Decimal(0)  # Final temperature of Battery (will be solving for this )
    t_a = Decimal(25) # Ambient temperature
    c = Decimal(902)  # Specific heat capacity in Joules per kg celsius
    k = Decimal(.0017)     # Cooling Constant (very roughly estimated from other papers)
    time_array = np.array([])
    temp_array = np.array([])
    t_change = 0
    
    for t in range(1,time):
        t_heating =(((current ** 2) * r)/m)/c  # Heating through internal resistance
        t_cooling =(k*(t_i-t_a))  #newtons cooling
        t_f = t_i + (t_heating - t_cooling)
        t_i = t_f
        time_array = np.append(time_array, [t])
        temp_array = np.append(temp_array, [t_f])
        t+= 1

    return time_array, temp_array


time = 2400  # time (our iterable) in seconds
